弹幕插件/弹幕姬没显示怎么办

如果开启了弹幕/礼物/欢迎/关注类插件,但是OBS里面没有显示,请按下面的步骤分析:

  1. 检查房间号
    一般来说,我们的房间号都是一串数字, 如果用浏览器打开自己的房间,那么最后的那串数字就是你的房间号,当然有些平台把那串数字当作是主播的ID号,但是在智播这边统一认为他是房间号.

房间号请务必保证正确,且一般来说都是数字,不可能是汉字!!!

  1. 关闭 Windows Defender 防火墙
    若要关闭 Windows Defender 防火墙，请选择“开始” 按钮，然后依次选择“设置” >“更新和安全” >“Windows 安全中心”>“防火墙和网络保护”。选择网络配置文件，然后在“Windows Defender 防火墙”下，将设置切换为“关”。

  2. 关闭路由器防火墙
    一般路由器都是有一个地址可以访问的,可以通过浏览器修改防火墙设置

    1. 打开浏览器,输入192.168.0.1或者192.168.1.1,如果都不对,就需要查询下(说明书或者网上搜索)该型号路由器的地址
    2. 一般来说用户名和密码都是admin 密码admin或者密码为空,如果实在不记得,路由器上有一个reset按键,他会恢复默认设置.
    3. 登录进去后,找到安全设置,防火墙设置,选择关闭即可
